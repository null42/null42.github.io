# 学习记录 #0015：后备式UPS完整软件设计——从零到一的固件实战

**日期**：2026-06-13
**课程**：第15课 — 后备式UPS完整软件设计（Phase 3系统集成）
**状态**：进行中

## 关键洞察

1. **后备式架构**：市电直通+电池后备，逆变器只在市电异常时工作
2. **4状态机**：INIT→STANDBY→INV_ON→FAULT，状态驱动+ISR控制
3. **市电检测**：ADC(幅值)+光耦(有无)，快切慢回消抖(20ms/2000ms)
4. **切换逻辑**：先断后合（后备式不做市电同步，先合后断会环流熔焊）
5. **逆变器启动**：5步时序，软启动200~500ms
6. **过载保护**：反时限特性，110%→10min, 150%→10s, 200%→立即
7. **前后台架构**：ISR 10kHz控制+1kHz检测+主循环状态机
8. **电机对照**：与电机控制器状态机高度相似，差异在切换触发和继电器延时
