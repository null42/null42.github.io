# 学习记录 #0019：通信协议与远程监控——UPS的"神经系统"

**日期**：2026-06-13
**课程**：第19课 — 通信协议与远程监控（Phase 3系统集成）
**状态**：进行中

## 关键洞察

1. **3种通信接口**：Modbus RTU(RS485远程监控) + CAN(并机通信) + RS232(本地维护)
2. **Modbus RTU帧边界**：3.5字符静默时间检测，不能用帧头帧尾（数据域不可预测）
3. **寄存器映射**：输入寄存器(只读30001~) + 保持寄存器(读写40001~)，定点数表示法(×0.1/×0.01)
4. **SNMP Trap**：事件驱动主动上报，比GET轮询延迟低（秒级vs分钟级）
5. **CAN并机**：500kbps，采样点87.5%，120Ω终端电阻
6. **OTA升级**：A/B双区+XMODEM传输+CRC32校验+签名验证+防降级
7. **安全关键**：写寄存器必须验证范围，固件必须签名验证
