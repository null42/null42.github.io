# 学习记录 #0007：单相PFC整流器——从整流到功率因数校正

**日期**：2026-06-13
**课程**：第7课 — 单相PFC整流器（Phase 2核心拓扑开始）
**状态**：进行中

## 关键洞察

1. **PFC目标**：PF>0.99，THD<5%，满足IEC 61000-3-2谐波标准
2. **双闭环结构**：电压外环（10~20Hz慢环）+ 电流内环（2~5kHz快环）
3. **电流参考生成**：Iref = A × Vac / V²rms，V²rms前馈消除输入电压变化影响
4. **电压环低带宽原因**：输出电压含100Hz纹波，带宽过高会将纹波耦合到电流参考→3次谐波→THD恶化
5. **非线性PI**：误差小用小增益（低THD），误差大用大增益（快速恢复），解耦两个矛盾需求
6. **占空比前馈**：dFF = 1 - Vin/Vout，PI只需补偿动态误差，大幅提高跟踪速度
7. **V²rms计算**：滑动窗口法，1300点环形缓冲区，每PWM周期更新

## 代码要点

- V²rms滑动窗口：环形缓冲区+累加和，O(1)复杂度
- 非线性PI：根据误差大小切换Kp/Ki
- 完整PFC控制：采样→V²rms→电压环→Iref生成→电流环→前馈→PWM
