# 学习记录 #0006：ADC采样技术——从传感器到数字量

**日期**：2026-06-13
**课程**：第6课 — ADC采样技术
**状态**：进行中

## 关键洞察

1. **采样信号全景**：电流（最高优先级）→ 母线电压 → 输出电压 → 电池 → 温度（最低）
2. **电流采样选型**：<50A用分流电阻（精度高成本低），>50A用霍尔传感器（隔离无损耗）
3. **低侧电阻采样**：10mΩ合金电阻 + 16.5倍差分运放 + RC滤波，Kelvin四端接法
4. **PWM-ADC同步**：中心对齐PEAK点触发ADC，下管导通中点采样，延迟0.5Tsw
5. **三相必须同步**：不同步→Clarke/Park后2倍频纹波→控制振荡
6. **零偏校准**：PWM未输出时采样256次取平均，运行时减去零偏
7. **过采样**：4×→+1位，16×→+2位，只适用于慢变信号，不适用于电流环

## 代码要点

- 零偏校准：PWM关断时256次平均，运行时减偏移
- C2000配置：ePWM SOCA触发 → ADCA SOC0/1/2 → DMA搬运 → ISR执行PI
- 过采样：N次采样求和右移log2(N)/2位
